## 一、项目是什么（一句话）

基于**本单位 200 例真实世界数据 + 30 例前瞻性 fMRI**，研究**经皮球囊压迫术（PBC）治疗三叉神经痛术后面部麻木**的危险因素、动态恢复轨迹、预测模型和中枢机制。

**核心主线**（贯穿全文，务必记住）：
```
临床现象（麻木） → 数学预测（列线图） → 神经机制（fMRI）
```

## 二、三部分研究结构

| 部分 | 设计 | 样本 | 目标 | 核心方法 |
|------|------|------|------|----------|
| **第一部分** | 回顾性队列 | 200 例 | 找规律：危险因素 + 麻木动态轨迹 | Logistic 回归 + **GBTM 轨迹建模** |
| **第二部分** | 建模 + 内部验证 | 同 200 例 | 做预测：术后面部麻木预测模型 | **LASSO + 列线图** + Bootstrap 验证 |
| **第三部分** | 前瞻性自身对照 | 30 例 | 找机制：麻木的中枢机制 | **静息态 fMRI**（ALFF/ReHo/功能连接/ICA） |

**关键判定阈值**（易混淆，单列）：
- 疼痛复发 = 术后 12 月 **BNI 疼痛 ≥ Ⅲ 级**
- 重度麻木 = **BNI 麻木 ≥ Ⅲ 级**（敏感性分析结局）
- 预测模型主结局 = **BNI 麻木 ≥ Ⅱ 级**
